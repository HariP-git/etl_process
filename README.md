# 🚗 Used Cars Price Prediction & Analysis

A complete Data Analytics + Machine Learning pipeline for analyzing used car data, performing visualization, feature engineering, and predicting car prices using Linear Regression and Random Forest Regressor.

---

# 📌 Project Overview

This project demonstrates an end-to-end workflow for:

- Data Extraction
- Data Cleaning
- Feature Engineering
- Exploratory Data Analysis (EDA)
- Data Visualization
- Machine Learning Model Training
- Model Evaluation
- Feature Importance Analysis

The dataset contains information about used cars such as:

- Mileage
- Engine Capacity
- Power
- Fuel Type
- Transmission
- Owner Type
- Manufacturing Year
- Price

The goal is to predict the selling price of used cars.
